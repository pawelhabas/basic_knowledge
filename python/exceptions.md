# Exceptions

## Built-in exceptions
There are many types of Exceptions. Here are some of them

- [AttributeError](attribute_error.py)
  - Occurs when you try to access not existing attribute of object

- [ImportError](import_errors.py)
  - Occurs when you try to import element not existing in specific location

  - [ModuleNotFoundError](import_errors.py)
    - Occurs when you try to import module that not exist

- [IndexError](index_errors.py)
  - Occurs when you try to access out of range element

- [KeyError](key_errors.py)
  - Occurs when you try to access element of dict that not exist 

- [NameError](name_errors.py)
  - Occurs when you try to use the name that was not defined 

- [NotImplementedError](not_implemented_error.py)
  - Occurs when you implement it. For example in not finished methods or functions

- [StopIteration](stop_iteration_error.py)
  - Occurs when you iterate more than rage
